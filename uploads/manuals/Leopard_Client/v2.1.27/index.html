1.2 1. 개요

1.2.1 1.1 소개

Leopard Client는 저장장치와 통신하여 영상 관제를 목적으로 사용합니다. - 데이터베이스와 통신하여 접속된 모든 영상을 관제 - 소규모부터 중·대형급까지 제한 없이 관제 가능 - 카메라 제어 - 녹화 영상 검색 및 백업

1.2.2 1.2 주요 특징

  • 저장분배서버를 통한 실시간 영상 모니터링
  • 소규모~중·대형급 모니터링
  • Leopard 장비의 편리한 접속
  • 실시간 모니터링 및 PTZ 카메라 원격 제어
  • 그래픽/사운드 알림으로 장애 시 관리자 인식률 상승
  • USB형 조이스틱 지원
  • Leopard 저장장치 녹화 영상의 검색/재생
  • 다양한 분할 화면 모드 및 사용자 분할 화면
  • 듀얼 라이브화면
  • 순차 전환이 가능한 분할 화면 모니터링
  • 즐겨찾기 채널/화면 구성을 그룹으로 관리
  • 다양한 알람 입력 연동 및 자동 알람 이벤트 처리
  • 이벤트 발생 시 팝업/알람음/경광등 연동
  • 백업 가능(뷰어 내장/AVI)
  • 편리한 단축키 제공
  • 동시 재생 화면 동기화
  • 듀얼 스트리밍으로 효율적인 성능 관리
  • 오디오 방송(채널·그룹별 스케줄)
  • H.264/H.265, 16/32채널 실시간 영상 표출
  • ONVIF, RTSP 프로토콜 지원
  • 접속한 VMS/NVR 영상의 실시간 통합 관제/제어
  • CPU/Memory/Network 사용량 모니터링

1.2.3 1.3 시스템 다이어그램

  • NVR 구성 (도식 참조)
  • VMS 구성 (도식 참조)

1.3 2. 설치하기

1.3.1 2.1 프로그램 설치

  1. 설치 프로그램 Leopard_Client_x64_Install_v2.1.xxxx 실행
  2. 설치 경로 선택 → 다음(N)
  3. 시작 메뉴 폴더명 입력(기본 Leopard) → 다음(N)
  4. 바탕화면 바로가기 생성 여부 선택 → 다음(N)
  5. 설치(I) 클릭하여 설치 진행
  6. 완료 후 종료(F). “Client 실행” 체크 시 곧바로 실행

(그림: 설치 마법사 화면들)

1.3.2 2.2 실행하기

  • 바탕화면 아이콘 실행 (Client PC 구동 시 자동 실행 가능)

1.3.3 2.3 인증키 삽입

  • 동봉된 USB 인증키를 PC USB 포트에 삽입해야 정상 동작
  • 인식 실패 시 프로그램 재시작 후 확인
  • 인증 실패 시 전자키 등록 메뉴/안내 팝업 확인

1.3.4 2.4 비밀 번호 입력

  • 비밀번호 및 서버 주소 입력(기본 비밀번호 admin, 사용 전 반드시 변경 권장)
  • 서버 주소
    • VMS: 관리서버 IP
    • NVR: NVR IP

1.3.5 2.5 데이터베이스 접속 실패 조치사항

  • 연결 실패 팝업 시 예(Y) 선택 → DB 입력 창
    • 주소: 관리서버/NVR IP
    • 포트: 기본 3306
    • 아이디: 기본 root
    • 패스워드: DB 설치 시 설정 값

1.4 3. 화면 구성 및 기능

1.4.1 3.1 라이브 화면

  1. 툴바: 라이브/재생/설정/상태 등 기능 선택
  2. 상태 표시창: 하드웨어 사용량 실시간 표시
  3. 환경설정 및 정보
  4. 제어 패널: 장치 목록, PTZ, 순차표출, 사용자 레이아웃, 이벤트 리스트
  5. 뷰 화면: 영상 분할 상태 지정
  6. 라이브 컨트롤: 디지털 줌, PTZ, 오디오, 캡처, 북마크 등
  7. 화면 분할: 분할 상태 지정

1.4.2 3.2 재생 화면

  1. 툴바(기능 선택)
  2. 카메라 선택 창(장치 목록)
  3. 재생 화면 창(1/4/9/16 분할)
  4. 연·월 선택
  5. 시·분 선택
  6. 재생 컨트롤(정/역·일시정지·속도 등)
  7. 간편 조작/분할 바(디지털줌/오디오/북마크/캡처)

1.5 4. 카메라 모니터링

  • 라이브 아이콘 선택 → 실시간 모니터링 화면

1.5.1 4.1 실시간 영상 보기

  • 장치 목록의 카메라/그룹을 라이브 화면으로 드래그
  • 장치/그룹에서 마우스 오른쪽 → 카메라 연결/그룹 연결

1.5.2 4.2 실시간 영상 끊기

  • 선택 창 우클릭 → 카메라 끊기
  • 전체 끊기 / 탭 끊기 / 카메라 끊기

1.5.3 4.3 PTZ 카메라 제어하기

  • 라이브 화면 우측 상단에 “P” 표시된 카메라만 제어 가능
  • PTZ 컨트롤:
    • 8방향 제어
    • 렌즈 제어(Zoom, Focus)
    • AUX 제어
    • 속도 슬라이더
    • 프리셋 저장/호출, 프리셋 투어(선택 채널/전체)

1.5.4 4.4 순차표출 사용하기

  • 종류
    • 생성된 순차표출 시작(탭 기반)
    • 그룹 내 순차표출 시작
    • 레이아웃 내 순차표출 시작
  • 화면 전환 시간: 환경설정 → 화면설정 → 순차표출

1.5.4.1 4.4.1 그룹 내에서 순차표출

  • 카메라 그룹/Live 화면에서 우클릭 → 원하는 순차표출

1.5.4.2 4.4.2 순차표출 추가 및 수정

  • 좌측 리스트 순차표출 메뉴 우클릭 → 추가
  • 이름/종류(카메라 그룹 또는 사용자 레이아웃)/전환 대기시간/분할 레이아웃 지정
  • 리스트 항목 우클릭 → 추가/수정

1.5.4.3 4.4.3 순차표출 시작 및 종료

  • 리스트 항목 우클릭 → 순차표출 시작
  • 종료: 해당 항목 우클릭 → 순차 표출 종료
  • 삭제: 항목 우클릭 → 삭제

1.5.5 4.5 사용자 레이아웃

  1. 추가: 현재 분할 상태를 레이아웃으로 저장
  2. 수정: 이름 수정/현재 레이아웃으로 업데이트
  3. 삭제
  4. 레이아웃 보드 표시/숨기기/설정(텍스트 표출 공간)

1.5.6 4.6 이벤트 리스트

  • 실시간 이벤트 모니터링
  • 이벤트 리스트 우클릭 메뉴:
    1. 라이브 팝업
    2. 재생 전환
    3. 이벤트 팝업 재생
    4. 옵션(이벤트 종류 선택)
  • 재생 중 카메라 필터링 기능

1.5.7 4.7 화면 분할 사용하기

1.5.7.1 4.7.1 전체 화면

  • 전체화면 아이콘 → Esc로 해제

1.5.7.2 4.7.2 화면 분할

  • 대칭형: 4/9/16/25/36/49/64분할(7종)
  • 비대칭형: 6A/12A/6B/8/12/7/12B/13A/13B/20/24(11종)
  • 사용자 분할 화면 지원

1.5.7.3 4.7.3 화면 전환

  • 예: 등록 16대, 4분할 → 페이지 라벨(1/N) 클릭 시 네비게이션 컨트롤 팝업 → 레이아웃 이동

1.5.8 4.8 라이브 컨트롤

1.5.8.1 4.8.1 디지털 줌

  • 카메라 선택 → (디지털 줌) 버튼 → 영역 지정/확대, 더블클릭 또는 아이콘으로 해제

1.5.8.2 4.8.2 PTZ 제어

  • (PTZ) 버튼 → Pan/Tilt/Zoom 제어 (속도 1~10)

1.5.8.3 4.8.3 오디오 수신

  • 카메라 선택 후 (오디오 수신) 버튼

1.5.8.4 4.8.4 오디오 송신

  • 카메라 선택 후 (오디오 송신) 버튼 → 마이크 입력

1.5.8.5 4.8.5 화면 캡쳐

  • 라이브/검색 화면에서 (화면 캡쳐)

1.5.8.6 4.8.6 북마크 입력

  • (북마크) 버튼 → 메모 입력 → 녹화영상에 저장 → 검색/재생 가능

1.5.8.7 4.8.7 상태 표시창

  • CPU(%), Memory(GB), Network(Mbps) 사용량 실시간 표시

1.5.8.8 4.8.8 라이브 영상 우클릭 메뉴

  • 전체/탭/카메라 끊기
  • 순차표출
  • 백업
  • 화면 캡쳐
  • 북마크 추가
  • PTZ 설정
  • 영상 비율 맞춤
  • 모션 설정(감지 영역)
  • 웹 페이지 열기(카메라)
  • Ping 테스트(네트워크 상태)
  • 카메라 정보(복사 가능)
  • 전체 화면

1.6 5. 녹화 영상 검색 및 재생하기

  • 저장분배서버/NVR 녹화 영상 검색/재생 (권한 필요)

1.6.1 5.1 검색

1.6.1.1 5.1.1 달력으로 검색하기

  1. 카메라를 재생 창으로 드래그 또는 우클릭 카메라 연결
  2. 달력 패널에서 날짜 선택
  3. 하단 시간 슬라이더로 시/분 선택

1.6.1.2 5.1.2 이벤트로 검색하기

  • 이벤트 검색 패널 활성화 → 이벤트 리스트에서 선택 → 더블클릭으로 재생화면 배치
  • 검색 옵션 선택 후 불러오기 → 재생

1.6.1.3 5.1.3 북마크 검색/삭제

  • 기간(시작~끝) 지정 → 불러오기 → 리스트에서 재생
  • 삭제: 리스트 항목 우클릭 → 삭제

1.6.1.4 5.1.4 파노라마로 검색하기

  • 카메라와 시간대 선택 → 파노라마 버튼 → 16분할 표출
  • 간격(1장~10분) 선택 후 화살표로 탐색
  • 움직임 감지 사용 시 움직임 발생 구간만 검색

1.6.2 5.2 재생

1.6.2.1 5.2.1 카메라 선택 및 선택 취소

  • 분할 선택(1/4/9/16)
  • 장치목록에서 드래그/더블클릭으로 배치 (동시 최대 16대)
  • 취소: 장치목록 우클릭 카메라 끊기 또는 더블클릭 / 화면에서 우클릭 카메라 끊기

1.6.2.2 5.2.2 재생하기

  • 재생 버튼
  • 속도: 0.5×, 1~8×, 16×, 32×
  • 스텝 재생: 1장, 1초~10분(총 11단위)
  • 시간 표시: 년/월/일/시/분/초/1/1000초 (동시 재생 동기화)

1.6.3 5.3 백업

  • 권한 필요, 뷰어 내장 백업 / AVI 파일 백업 지원

1.6.3.1 5.3.1 백업 설정

  • 마우스 우측클릭 → 백업
  • 날짜/시간: (1) 시작~끝, (2) 시작으로부터 기간(일/시/분/초)
  • 백업 타입: AVI / 뷰어 내장
  • 파일명(저장 경로), 이름 폴더 생성 옵션
  • 뷰어 내장 백업: 비밀번호 설정 가능

1.6.3.2 5.3.2 백업 영상 검색

  • 생성된 백업 파일 더블클릭 → Backup Viewer 실행
  • 비밀번호 설정 시 인증 창 선행

1.6.3.3 5.3.3 백업 영상 재생

  • 검색/재생 화면 조작 방식과 유사
  • 배속 재생 시 모든 프레임 재생
  • 타임바 클릭 시 해당 구간으로 이동

1.7 6. 설정

  • 관리서버/NVR 접속 후 카메라/그룹 운영·관리
  • 가져오기: admin(기본) 또는 사용자 선택

1.7.1 6.1 설정 화면

  • 툴바 → 설정 아이콘

1.7.2 6.2 PTZ 카메라 설정

  • 설정 → PTZ
  • 카메라 선택, 프리셋 추가/수정/삭제
  • 홈 프리셋(무입력 시간 경과 시 이동)
  • 프리셋 투어(번호/지속시간)

1.7.3 6.3 이벤트 설정

  • 입력/동작/맵 3가지를 모두 설정해야 정상 동작

1.7.3.1 6.3.1 이벤트 설정(입력)

  • 리스트/메뉴(추가·수정·삭제·모두 삭제)/필터

1.7.3.2 6.3.2 이벤트 입력

  • 추가 → 이벤트 이름 → 이벤트 종류(예: Camera) → 카메라 선택(단일/전체) → 체크 후 추가

1.7.3.3 6.3.3 이벤트 수정

  • 항목 선택 → 수정 → 값 변경 후 저장

1.7.3.4 6.3.4 이벤트 삭제 및 모두 삭제

  • 항목 삭제, 전체 모두 삭제(재확인)

1.7.3.5 6.3.5 이벤트 동작

  • 추가 → 이벤트 이름 → 동작 종류 선택 → 클라이언트 IP/포트 → 추가

1.7.3.6 6.3.6 이벤트 동작: Popup

  • 팝업 화면: 1x1 또는 2x2
  • 카메라 선택(1x1: 1대, 2x2: 최대 4대)
  • 새 팝업 윈도우 / 최상위 / 자동 닫힘 / 닫힘 시 프리셋 이동 / 저해상도 사용

1.8.4.1 7.4.1 영상

  • 빠른 스트림 전환 사용(고/저해상도 전송 방식)
  • 선명하게 하기(고품질, CPU 사용량 고려)
  • 영상 끊김 시 마지막 영상 유지
  • Direct3D 하위버전/하드웨어 코덱(환경에 따라 자동)
  • 버퍼링 타입: 적응지연(기본)

1.8.4.2 7.4.2 OSD / 순차표출

  • 글자 크기(기본 16) / 색상(기본 백색)
  • 배경(테두리) 색상(기본 검정) / 선 두께(기본 1)
  • 다음 시간 후 연결 중 표시(기본 10초)
  • 전체화면 날짜/시간 표시 위치
  • 순차표출 지속 시간(기본 10초)
    • 주의: 그룹 내 순차표출은 전체 화면에서는 동작하지 않음(분할 이하에서 동작)

1.9 8. 문제 해결 방법(FAQ)

  • 영상이 “연결 중” 상태
    1. 라이브 화면에서 해당 카메라 Ping 테스트
    2. Ping 불가: 네트워크 문제 가능(관리자 문의)
    3. Ping 정상: 카메라 설정 문제 가능 → 카메라 웹 페이지에서 설정 확인
    4. 상태에서 카메라 사용 안 함(X) 여부 확인 → 사용함으로 변경
    5. 고해상도/저해상도/녹화에 X 표시: 네트워크 문제 가능(동일 조치)
  • 다수/전체 영상이 접속 중
    • 모니터링 장소의 네트워크 상태 문제 또는 저장분배서버(NVR/CMS) 문제 가능 → 담당 업체 문의
